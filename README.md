# HalloweenEssentials
A halloween plugin for 10b10t.org.

## What it does
Funny things happen when right/left clicking with a pumpkin in your hand.

## Funny things
* Levitation
* Slow effect
* Speed effect
* Jump boost
* Glowing
* Fire resistance
* Funny messages
* Lightning
* Change weather
* Primed tnt
* Crystals
* Enchanted golden apples
* Shulkers (Filled with delicious cookies)
* Totems
* Friend wolfs
* Witches
* Blazes
* Skeletons
* Zombies
* Change weather
* Jokes no one gets
* Get special textured heads
* Get a tasty cookie
