# HalloweenEssentials
A halloween plugin for 10b10t.org.

## What it does
Funny things happen when right/left clicking with a pumpkin in your hand.

## Funny things
* Levitation
* Slow effect
* Speed effect
* Jump boost
* Glowing
* Fire resistance
* Funny messages
* Lightning
* Change weather
* Primed tnt
* Crystals
* Gapples
* Shulkers
* Totems
* Friend wolfs
* Witches
* Blazes
* Skeletons
* Zombies
* Change weather
* Jokes noone gets
* Get special textured heads
