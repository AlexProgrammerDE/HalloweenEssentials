# HalloweenEssentials
A halloween plugin for 10b10t.org.

## What it does
Funny things happen when right/left clicking with a pumpkin in your hand.

## Funny things
* Potion effects
* Funny messages
* Lightning
* Change weather
* Spawn tnt
* Crystals
* Gapples
